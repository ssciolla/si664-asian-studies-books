Asian Studies Books Project

This site presents descriptive metadata and library holdings statistics for 365 titles published by centers of Asian studies on the University of Michigan campus over the last several decades. As part of a National Endowment for the Humanitities-funded project, Michigan Publishing Services will create enriched, open access versions of one hundred of these titles.

The Books tab provides a list of titles. The Filter Search tab allows for more complex searching and filtering. The About tab provides more details about the project.

Code for the project can be found at https://github.com/ssciolla/si664-asian-studies-books.
